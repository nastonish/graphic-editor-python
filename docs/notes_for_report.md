# Нотатки до підсумкового звіту

## 1. Мета проєкту
- Описати навчальну мету графічного редактора.
- Пояснити зв'язок із лабораторними роботами 8, 9, 10.

## 2. Зовнішній інтерфейс графічного редактора
- Загальний вигляд головного вікна.
- Опис меню, панелі інструментів, полотна, рядка стану.
- Пояснення логіки українських підписів команд.

## 3. Підсистема довідкової інформації
- Наявність зовнішнього файлу `help.html`.
- Пункт меню для відкриття довідки у браузері.
- Вікно "Про програму".

## 4. Папка прикладів і коментарів
- Призначення папки `examples/`.
- Опис файлу `examples/comments.txt`.
- Перелік запланованих демонстраційних зображень.

## 5. Функціональні групи команд і параметрів
- Група "Файл".
- Група "Редагування".
- Група "Інструменти".
- Група "Зображення".
- Група "Вигляд".
- Група "Довідка".

## 6. Реалізовані можливості (поточний етап)
- Створено базове вікно застосунку.
- Додано робочі інструменти малювання: олівець, лінія, прямокутник, еліпс.
- Додано полотно із горизонтальною та вертикальною прокруткою.
- Реалізовано малювання на Pillow-зображенні з відображенням на полотні Tkinter.
- Додано вибір кольору лінії, кольору заливки, товщини лінії, скасування Ctrl+Z та Esc для скасування незавершеного попереднього перегляду.

## 7. Заплановані наступні кроки
- Реалізація інструментів малювання.
- Реалізація операцій редагування (скасування, виокремлення, копіювання, вставлення).
- Реалізація операцій обробки зображення.
- Підтримка відкриття/збереження файлів.

## 8. Скріншоти для звіту
- Головне вікно програми.
- Приклад малювання фігур.
- Приклад виокремлення та вставлення.
- Приклад застосування фільтрів.
- Приклад роботи з великим зображенням і прокруткою.

## 9. Висновок
- Підсумувати відповідність вимогам викладача.
- Описати отримані практичні навички.
