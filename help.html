Довідка: Графічний редактор

Навчальний графічний редактор для створення та редагування простих растрових зображень.

Призначення програми

Програма дозволяє створювати нові зображення, малювати фігури, виокремлювати області, копіювати фрагменти, вставляти їх і зберігати результат у файл.

Основні розділи інтерфейсу

  • Меню команд у верхній частині вікна.
  • Панель інструментів для швидкого доступу.
  • Полотно з горизонтальною та вертикальною прокруткою.
  • Рядок стану з інформацією про поточний інструмент і дії.

Інструменти малювання

Доступні інструменти: Олівець, Лінія, Прямокутник, Еліпс, Виокремити.

  • Для лінії, прямокутника, еліпса та виокремлення показується тимчасовий попередній перегляд під час перетягування.
  • Для олівця малювання відбувається короткими сегментами під час руху миші.
  • Доступний вибір кольору лінії, кольору заливки та товщини лінії.

Файлові операції

  • Новий (Ctrl+N) — створює нове біле зображення.
  • Відкрити (Ctrl+O) — відкриває PNG, JPG/JPEG або BMP.
  • Зберегти (Ctrl+S) — зберігає поточне зображення.
  • Зберегти як — зберігає копію з новою назвою.
  • Вийти — закриває програму.

Операції редагування

  • Скасувати (Ctrl+Z) — повертає попередній стан зображення.
  • Виокремити — виділення прямокутної області мишею.
  • Копіювати — копіює виокремлений фрагмент у пам'ять.
  • Вставити — вставляє скопійований фрагмент на полотно.
  • Очистити виділене — заповнює виокремлену область білим кольором після підтвердження.
  • Esc — скасовує поточний незавершений попередній перегляд (фігури або виокремлення).

Якщо область не виокремлена, для команд копіювання та очищення виводиться повідомлення: «Спочатку виберіть область зображення.»

Якщо немає скопійованого фрагмента, для вставлення виводиться повідомлення: «Немає скопійованого фрагмента.»

Типовий сценарій роботи з виокремленням

  1. Оберіть інструмент Виокремити.
  2. Затисніть ліву кнопку миші та протягніть прямокутник.
  3. Відпустіть кнопку — область буде збережена як виділена.
  4. Натисніть Копіювати, потім Вставити для дублювання фрагмента.
  5. За потреби використайте Очистити виділене.
